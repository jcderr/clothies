## <a name="setup"></a> Setup

#### Global Dependencies

```bash
# install bower globally
sudo npm install -g bower

# install global nodemon
sudo npm install -g nodemon

# install global gulp
sudo npm install -g gulp-cli
```


#### Project Dependencies install

```bash
# install npm modules for project
npm install

# install bower packages for project
bower install
```



## <a name="localdev"></a> Local Dev

#### Run local server

```bash
# start gulp process, this will watch for changes and auto update files and auto restart server. just refresh browser after changes are saved√Ç
gulp
```
